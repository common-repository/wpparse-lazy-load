Lazy Load images in post/page content and featured images.

== Description ==

The best lazy load plugin by WPPARSE to load images fast. Options to loazy load images in post/page content and featured images. WPPARSE Lazy Load is a fastest and lightest solution to load images fast. This plugin adds an attribute "data-originial" with the image URL in the post/page content and featured images on the front-end of a WordPress website.

[WPPARSE Lazy Load Demo](http://www.wpparse.com/lazy-load/) 

== Installation ==

1. Go to Plugins -> Add New, and search for 'WPPARSE Lazy Load'. Install and activate it.
2. Download the zip file from the WordPress plugin repository. Go to Plugins -> Add New. Click 'Upload Plugin' on the top and upload.
3. Manually upload the plugin files to the '/wp-content/plugins/wpparse-lazy-load' directory. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How to enable lazy load for images in post / page content? =

Go to 'Lazy Load' menu and check the 'Lazy Load on Posts / Pages' checkbox.

= How to enable lazy load for featured images? =

Go to 'Lazy Load' menu and check the 'Lazy Load on Featured Images' checkbox.
